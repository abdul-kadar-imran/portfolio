# Modern React Portfolio (Vite + React Router + Framer Motion)

## Quick Start
```bash
npm install
npm run dev
```

Pages: Home, About, Skills, Projects, Certifications, Resume, Contact.

- Routing: `react-router-dom@6`
- Animations: `framer-motion`
- Build tool: `Vite`
- Styling: vanilla CSS with modern variables and responsive grid.

Update images in `src/assets/images/` and replace `src/assets/resume.pdf` with your real resume.
